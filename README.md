# fullstack

## How start project on Windows:

```
git clone https://github.com/s2khan6/fullstack

cd fullstack

pip -r requirements.txt

python manage.py runserver
```

## For Linux and Mac os

```
git clone https://github.com/s2khan6/fullstack

cd fullstack

pip3 -r requirements.txt

python3 manage.py runserver
```
